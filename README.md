# Triathlon Time Calculator

Free triathlon time calculator for swimming, biking, and running.

## Quick Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Swimming: 300m to 3800m
- Biking: 20km to 180km
- Running: 5km to 42km
- Keyboard navigation with arrow keys
- Fully responsive
- SEO optimized
- PWA ready

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React
